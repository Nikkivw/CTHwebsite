Cheatsheet

• Maak een lege map aan
• Zet er een README.md bestand in
• In Terminal typ “git init” om git te initializen
• Voeg je bestand toe aan de tracker met “git add README.md”
• Voeg een branch toe aan je git met “git branch -M main”
• Voeg een remote aan je git toe genaamd origin met “git remote add origin {jouw GitHub repository.git}
• Commit de veranderingen met git commit -m “message”
• Push je git repository naar github met “git push -u origin main”
• En tadaaa
